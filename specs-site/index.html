A.R.C. Platform — Specs ​

This is the searchable spec documentation site for the A.R.C. Platform. It publishes all feature specifications, implementation plans, and task breakdowns from the specs/ directory as a statically rendered site with full-text search and mermaid diagram support.

Local Development ​

bash
cd docs/specs
npm install       # first time only
npm run dev

The dev server starts at http://localhost:5173/arc-platform/specs-site/ with live reload enabled.
